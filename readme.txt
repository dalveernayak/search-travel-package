== Description ==

You can you to add Flights, Hotels, Vacations and Car Rentals and also can search by adding shortcode in page and posts




Copy one of the following shorcodes into any post inside Wordpress visual editor:

*[nayak_searchboxes options='{"size":"dynamic"}']*

*[nayak_searchboxes options='{"size":"160x600"}']*

*[nayak_searchboxes options='{"size":"300x250"}']*

*[nayak_searchboxes options='{"size":"300x533"}']*

*[nayak_searchboxes options='{"size":"728x90"}']*

or simply hit the orange palm tree icon inside your wordpress editor, customize and add a searchbox into your post by a few clicks.

To add a searchbox to your header, footer or sidebar, open the template file in Wordpress editor and copy the following code in it (based on the sice you want to use):

*`<?php echo do_shortcode('[nayak_searchboxes options=\'{"size":"dynamic"}\']'); ?>`*

*`<?php echo do_shortcode('[nayak_searchboxes options=\'{"size":"160x600"}\']'); ?>`*

*`<?php echo do_shortcode('[nayak_searchboxes options=\'{"size":"300x250"}\']'); ?>`*

*`<?php echo do_shortcode('[nayak_searchboxes options=\'{"size":"300x533"}\']'); ?>`*

*`<?php echo do_shortcode('[nayak_searchboxes options=\'{"size":"728x90"}\']'); ?>`*
